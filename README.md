# Customer-Experience-Analytics-for-Fintech-Apps

This repository contains code and pipelines to:

- Scrape Google Play Store reviews for three Ethiopian bank apps
- Clean and preprocess reviews
- Run sentiment and thematic analysis
- Store results in PostgreSQL

Branches:

- task-1: data collection & preprocessing
- task-2: sentiment & themes
- task-3: database load & schema
- task-4: visualizations & report

See /src for scripts
